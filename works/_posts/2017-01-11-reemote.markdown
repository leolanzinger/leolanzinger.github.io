Reemote is a side project built with [Mohamed Osman][mo]. It's a search engine for live video streams on Youtube, Twitch and Periscope. With Reemote, users are able to search for videos that are being live streamed on these platforms and see an aggregate list that allows them to quickly look through multiple platform results.

The website is online and you can check it here: [reemote.herokuapp.com][reemote].

![reemote gif](/images/reemote/live.gif)
*The main screen of Reemote*

I worked on the design and frontend development of the website, which is build with React. Indeed, it was a really nice way to start learning [React JS][react]. Mohamed worked on the backend of the website. You can check out a nice blogpost about Reemote on [Mohamed's personal blog][mopost].
